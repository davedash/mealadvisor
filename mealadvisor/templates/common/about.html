Colophon

The original site (formerly "meal advisor") was built on the symfony PHP5 web framework and was used as a model for creating this current incarnation.

The site was redesigned using OmniGraffle and inspired by black-background sites such as Popurls. We used the Silk icons from Fam Fam Fam. Occasional assistance came from Adobe's Illustrator and Photoshop products.

We use Slicehost VPS to host a virtual Ubuntu server. We serve static content using nginx (pronounced "Engine-X") and requests for dyanmic content are delivered to an Apache 2 HTTP server running mod_python. The meal advisor application is built using the django web framework for python. Using mysql as its glue.

The site is developed for Firefox, Safari, Internet Explorer 7 and Internet Explorer 6.
